# EONBOT Command Center

EONBOT should help users control EONAPP by text and voice.

Safe commands can navigate or explain:

- open EON City
- show rewards
- open Vault
- open Market
- explain Sponsor Boost
- explain backups

Sensitive commands require confirmation:

- enable microphone
- enable Sponsor Boost / MultiTag
- payment proof
- wallet action
- backup mutation
- reward claim
